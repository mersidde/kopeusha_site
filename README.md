# 🌿 Сайт психолога Ксении Копейкиной

Веб-сайт для  психолога, телесного терапевта и коуча, специализирующегося на работе с взрослыми детьми деструктивных семей (ВДД).

## 🎯 О проекте

Сайт разработан как инструмент привлечения клиентов и представления услуг психолога. Включает информацию о специалисте, форматах работы, продуктах, а также форму записи на консультацию.

**Целевая аудитория:** взрослые люди (ВДД), выросшие в деструктивных семьях, испытывающие чувство вины, стыда, тревоги и трудности в построении отношений.

## ✨ Особенности

- 📱 **Адаптивный дизайн** — корректно отображается на всех устройствах
- 🎨 **Единый корпоративный стиль** — мягкая фиолетовая палитра, тёплые акценты
- ⚡ **Высокая производительность** — без внешних библиотек и фреймворков
- ♿ **Доступность** — семантическая разметка HTML5
- 🔒 **Безопасность** — honeypot-защита от спама, валидация данных
- 📝 **Умная форма записи** — маска телефона, живая валидация, счётчик символов
